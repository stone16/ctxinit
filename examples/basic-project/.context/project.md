# Basic Project

This is an example project demonstrating how to use ctxinit for managing AI context files.

## Overview

This project shows the basic setup and usage of ctxinit with:

- A simple configuration file
- Example rules for different use cases
- Multi-target compilation support

## Tech Stack

- TypeScript
- Node.js
- Jest for testing

## Getting Started

1. Install dependencies: `npm install`
2. Build context files: `ctx build`
3. Start development: `npm run dev`
